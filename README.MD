# Tanke Themes

A collection of carefully crafted dark themes for Visual Studio Code — from deep oceanic blues to neon purples and full cyberpunk aesthetics.

---

## Themes

### Abyss Blue

A deep, oceanic dark theme built on rich navy tones. Blue functions, lavender keywords, and steel-grey variables create a calm yet precise environment for long coding sessions.

| Element    | Color        |
|------------|--------------|
| Background | Deep navy    |
| Keywords   | Soft lavender |
| Functions  | Electric blue |
| Strings    | Ice cyan      |
| Types      | Warm amber    |
| Cursor     | Electric blue |

---

## Installation

### From the Marketplace

1. Open VS Code
2. Open **Extensions** panel (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for **Tanke Themes**
4. Click **Install**

### From a `.vsix` file

```bash
code --install-extension tanke-themes-x.x.x.vsix
```

### Selecting a theme

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Type `Preferences: Color Theme`
3. Select one of: **Abyss Blue**, **Velvet Amethyst**, **Neon Lilac**, or **Cyber Void**

---

## Building locally

Requires [vsce](https://github.com/microsoft/vscode-vsce):

```bash
npm install -g @vscode/vsce
vsce package
```
